Welcome to Game Of Life Multiplayer!

Click on cells to make them alive and paint them with the color assigned to you (highlighted user icon is you).

Game flows following these rules:

  1. Any live cell with fewer than two live neighbors dies, as if caused by under-population.
  2. Any live cell with two or three live neighbors lives on to the next generation.
  3. Any live cell with more than three live neighbors dies, as if by overcrowding.
  4. Any dead cell with exactly three live neighbors becomes a live cell, as if by reproduction.
  5. When a dead cell revives by rule #4, it will be given a color that is the average of its neighbors (that revive it).
